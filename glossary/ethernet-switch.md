# Ethernet Switch

## Definition

An Ethernet switch is a network device that connects multiple devices within a local area network (LAN) and forwards data to the correct destination based on MAC addresses. Switches are essential for efficient, high-speed communication in modern networks.

## How It Works

Switches receive data packets and examine their destination MAC addresses, then send the packets only to the intended device, rather than broadcasting to all devices. This reduces network congestion and improves performance. Managed switches offer advanced features like VLANs, QoS, and security controls.

## Real-World Application

Ethernet switches are used in offices, schools, data centers, and homes to connect computers, printers, servers, and other devices. They support everything from small workgroups to large enterprise networks.
