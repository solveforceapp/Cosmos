# Basic Rate Interface (BRI)

## Definition

Basic Rate Interface (BRI) is a type of Integrated Services Digital Network (ISDN) circuit that provides digital voice and data transmission over traditional telephone lines. BRI consists of two 64 kbps bearer channels (B channels) and one 16 kbps data channel (D channel), supporting simultaneous communication and signaling.

## How It Works

BRI enables users to make phone calls, send faxes, and transfer data using a single line. The B channels carry voice or data, while the D channel handles call setup, control, and signaling. BRI offers higher quality and reliability than analog lines, with features like caller ID, call forwarding, and conferencing.

## Real-World Application

BRI was widely used in small businesses, home offices, and remote locations before broadband became prevalent. It enabled early internet access, video conferencing, and integrated communication services. While less common today, BRI laid the groundwork for modern digital networks.
